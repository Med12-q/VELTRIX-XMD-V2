Comment lier :
1. Ouvre WhatsApp sur ton téléphone
2. Va dans Paramètres → Appareils liés
3. Appuie sur "Lier un appareil"
4. Sélectionne "Lier avec un numéro de téléphone"
5. Entre le code affiché ci-dessus

Le QR code se charge…
Si indisponible, utilise le code de couplage.

📖 Guide de déploiement
  1. Fork ce repo sur ton compte GitHub depuis github.com
  2. Crée un compte sur la plateforme de ton choix : Railway, Render, Heroku, Koyeb ou Replit
  3. Importe le repo GitHub dans la plateforme — elle détecte automatiquement Node.js
  4. Configure la variable d'environnement OWNER_NUMBERS avec ton numéro WhatsApp
  5. Démarre le déploiement — le bot démarrera et ce dashboard sera accessible
  6. Ouvre l'URL publique et entre ton numéro pour générer le code de couplage
  7. Entre le code dans WhatsApp → Appareils liés → Lier un appareil
  8. ✅ Le bot est maintenant actif et répondra aux commandes WhatsApp !
