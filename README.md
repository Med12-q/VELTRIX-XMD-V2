## 🔗 ÉTAPE 3 — CONNEXION WHATSAPP (Dashboard Web)

> Une fois déployé, ouvre l'URL publique fournie par la plateforme. Le dashboard VELTRIX XMD s'affiche.

```
1. Entre ton numéro WhatsApp (format international, sans +)
   Exemple : 224669288332

2. Clique sur "⚡ Obtenir le code de couplage"

3. Ouvre WhatsApp sur ton téléphone :
   Paramètres → Appareils liés → Lier un appareil
   → "Lier avec un numéro de téléphone"
   → Entre le code affiché sur le dashboard

4. ✅ Le bot est maintenant actif !
```

## 🔒 OBFUSCATION DU CODE

> VELTRIX XMD intègre un système de protection du code source. Lance `npm run build` pour générer une version obfusquée dans `dist/`.

```bash
npm run build     # génère dist/ avec code protégé
node dist/index.js  # lancer la version sécurisée
```

Protections appliquées :
- ✅ Encodage Base64 des chaînes
- ✅ Identifiants hexadécimaux
- ✅ Mélange des tableaux de chaînes
- ✅ Appels transformés & imbriqués
- ✅ Expressions numériques obscurcies
